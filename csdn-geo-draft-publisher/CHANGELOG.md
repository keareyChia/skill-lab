# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release
- Initialize CSDN GEO Draft Publisher Skill.
- Create prompt chain for reading, rewriting, formatting, and quality checking CSDN content.
- Implement Playwright-based browser automation script for safe, manual-review-required draft filling.
- Define strict safety rules, explicitly forbidding automated publication, cookie scraping, and credential harvesting.
- Setup example inputs based on PowerMatrix / OpenClaw use case and example outputs.
