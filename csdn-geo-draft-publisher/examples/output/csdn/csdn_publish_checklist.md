# CSDN 发布前人工审核清单

请在点击 CSDN 的“发布文章”按钮前，逐一核对以下事项，确保内容质量与合规性。

- [ ] **事实检查**：文章提及的技术原理、架构、产品功能（如：PowerMatrix / OpenClaw）是否真实准确，无虚假编造？
- [ ] **营销合规**：文章中是否有过度营销、拉踩竞品、包含“全网第一”、“最强”等绝对化用语？
- [ ] **技术价值**：文章是否具备实际工程参考价值（有清晰的输入输出、配置示例或架构图）？
- [ ] **敏感信息检查**：代码块或配置中是否包含了真实的 API Key、Token、私网 IP、真实用户手机号等隐私数据？
- [ ] **AI-GEO 承诺检查**：文章或摘要中是否**没有**夸大承诺（如“保证能被 DeepSeek 收录”等不实描述）？
- [ ] **社区规范检查**：文章排版是否整洁，是否符合 CSDN 技术社区的调性，而非营销软文？
- [ ] **发布确认**：我已确认该草稿无误，且发布行为是由我本人主动点击的。

> **提示**：如果以上任何一项不符合，请立刻在编辑器中修改。修改完成后再点击发布。
