# CSDN Automation Guidelines

This directory contains the rules, logic, and safety boundaries for the Playwright-based browser automation script used to fill CSDN drafts.

## Files
- `playwright_draft_flow.md`: Explains the step-by-step logic of the automation script.
- `safety_rules.md`: Strictly defines what the script CAN and CANNOT do, ensuring account safety and compliance.

## Core Principle
**Human-in-the-loop (HITL)** is mandatory. We automate the boring part (copy-pasting text into the editor), but the critical parts (review, verification, publication) must be done manually by a human.
