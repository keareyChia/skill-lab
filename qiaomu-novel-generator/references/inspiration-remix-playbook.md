# Inspiration Remix Playbook

Use this whenever the user gives a plot idea, asks for a more exciting story, mentions a famous film/TV/novel, or asks for an author-like style.

The default move is discussion plus selection:

1. Translate the user's idea into story functions.
2. If the user asks to search or the reference needs current/public context, run `source-research-remix.md` first.
3. Offer several famous reference beats as inspiration options.
4. Let the user choose or combine.
5. Build a new outline with new characters, setting, stakes, objects, rules, and ending.
6. Translate any author/style request into craft features, not direct imitation.

## Remix Boundary

Allowed:

- Use high-level story functions: fall from grace, comeback arena, mistaken identity, public reveal, impossible choice, long revenge, hidden proof.
- Mention famous works as reference signals.
- Combine 2-4 inspirations into a new premise.
- Change genre, setting, profession, power system, conflict arena, emotional payoff, and ending.
- Use broad craft traits such as dense historical detail, clipped dialogue, institutional pressure, black humor, lyrical sensory prose, or puzzle-box structure.

Not allowed:

- Copy protected passages, signature dialogue, character names, iconic objects, scene choreography, unique twists, or a recognizable sequence of events.
- Present the output as an exact continuation, sequel, alternate chapter, or same-world story of a protected work.
- Directly imitate the distinctive style of a living author. For living authors, convert the request to generic craft features.

Safe response for a living-author style request:

```text
我不能直接仿写在世作者的独特文风，但可以提取可泛化技法：制度缝隙、史料感细节、小人物在大系统里的奔走、轻讽刺、线索密集和节奏反转。我们用这些技法写一个全新的故事。
```

## Classic Beat Cards

Offer 3-6 cards per user premise. Do not dump the whole library.

### Downfall And Comeback

| Signal | Reusable function | Fresh transformation |
|---|---|---|
| 食神式 | 顶峰人物被背叛跌落，底层重新学会核心手艺，最终在公开竞技场回归 | 换行业、换技能、换代价；重点是“失去名号后重新理解技艺” |
| Rocky 式 | 弱者获得一次不对等挑战，通过训练证明尊严 | 换成修仙试炼、公开审查、黑市竞价、庭审质证 |
| Count of Monte Cristo 式 | 被陷害者长期隐忍，换身份归来，用对手欲望完成复仇 | 缩短为短篇时保留“证据、身份、诱饵、代价” |
| Jane Eyre 式 | 被轻视者守住自尊，拒绝不平等的爱，最终以完整人格回归 | 适合女性成长、虐恋反转、现实题材 |

### Public Proof And Face-Slapping

| Signal | Reusable function | Fresh transformation |
|---|---|---|
| 拍卖鉴宝式 | 众人误判价值，主角识破真伪，公开改变权力 | 可变成黑市灵物、公开质证、法庭证物、家宴礼物 |
| Courtroom reveal 式 | 所有人相信一个证词，主角用细节翻案 | 可变成宗门审判、直播舆论、公司复盘会 |
| Tournament trial 式 | 强者按旧规则碾压，主角理解规则漏洞获胜 | 可变成大比、考试、面试、试药、AI 评测 |
| Public decision reversal 式 | 主角被资历、规矩或话语权压制，用一线证据改变决策 | 适合组织逆袭、商业爽文和权力斗争 |

### Identity And Recognition

| Signal | Reusable function | Fresh transformation |
|---|---|---|
| Cinderella 式 | 被家庭系统低估的人，在公开场合被重新识别 | 重点不是王子，而是“身份、尊严、见证人” |
| The Prince and the Pauper 式 | 身份互换暴露阶层规则和真实能力 | 可变成修仙、豪门、学校、公司、太空殖民地 |
| The Bourne Identity 式 | 主角失去记忆，但身体/习惯暴露过去能力 | 可变成封印记忆、遗失功法、退役高手 |
| Mulan 式 | 主角隐藏身份进入禁区，以行动争取承认 | 可变成宗门禁地、军营、公司核心项目、家族议事 |

### Suspense And Truth Reframe

| Signal | Reusable function | Fresh transformation |
|---|---|---|
| Rashomon 式 | 多个证词都像真相，最后显示每个人都在保护自己 | 适合悬疑、家族秘密、宗门旧案 |
| Sixth Sense 式 | 读者一直误解叙事前提，结尾重读前文 | 只能借“前提反转”功能，不复刻生死设定 |
| Memento 式 | 记忆/记录不可靠，线索顺序本身是陷阱 | 适合记忆交易、重生漏洞、档案悬疑 |
| Sherlock 式 | 微小细节推翻大判断 | 适合推理、鉴宝、医案、组织真相 |

### Romance And Emotional Pull

| Signal | Reusable function | Fresh transformation |
|---|---|---|
| Pride and Prejudice 式 | 误判、阶层、傲慢和尊严慢慢反转 | 适合双强、组织关系、豪门、学院 |
| Beauty and the Beast 式 | 怪物/冷面身份背后有伤，亲密关系迫使双方改变 | 换成契约婚姻、禁忌搭档、诅咒关系 |
| 追妻火葬场式 | 一方误伤后迟来认知，必须付出行动代价 | 不要靠道歉解决，要靠选择和损失 |
| Bodyguard 式 | 保护关系里混入权力、误会和牺牲 | 适合甜宠、悬疑爱情、末世生存 |

### System And Bureaucratic Pressure

| Signal | Reusable function | Fresh transformation |
|---|---|---|
| institutional thriller 式 | 小人物被制度缝隙夹住，用文件、规则、时间差求生 | 适合历史、组织、官场、宗门规章、平台规则 |
| heist 式 | 团队分工、计划、意外、反计划 | 换成抢灵药、偷证据、救人、数据迁移、婚礼劫局 |
| Kafkaesque 式 | 规则荒诞但严密，主角越解释越陷入系统 | 适合黑色幽默、现实讽刺、恐怖 |
| The Big Short 式 | 少数人看见系统性错误，所有人都嘲笑，最后崩盘 | 适合商业、科幻、灾难预警 |

### Backstage Project And Organization Satire

| Signal | Reusable function | Fresh transformation |
|---|---|---|
| 后台项目重解式 | 熟悉的大事件不从英雄视角写，而从协调、背锅、补洞的人写 | 换成现代组织、宗门联盟、医院体系、太空迁移、家族企业 |
| Yes Minister 式 | 表面流程和真实权力错位，笑点来自话外之音 | 换成平台治理、审计会、资源分配会、长老议事 |
| Succession 式 | 继任权、忠诚测试、公开羞辱和派系下注纠缠 | 缩成一次投票、遗嘱、项目复盘、事故问责 |
| Spotlight 式 | 文件链条和小团队调查掀开被保护的真相 | 换成账册、日志、病历、案卷、拍卖目录 |
| 12 Angry Men 式 | 一个小疑点在封闭评议中推翻多数确定性 | 换成公开审查、宗门裁决、董事会、家宴质证 |
| The Office / Dilbert 式 | 荒诞规则和办公室小动作暴露系统荒唐 | 只借喜剧机制，必须加入真实 stakes 和人物欲望 |

## Style Signal Translation

When the user says `用某某风格`, do not copy the surface.

Translate into craft sliders:

| User style signal | Translate to |
|---|---|
| 马伯庸式 | 历史/制度细节，小人物卷入大系统，文件线索，轻讽刺，节奏密集，伏笔回收 |
| 金庸式 | 命运压力，江湖伦理，师承门派，情义债，人物选择有历史重量 |
| 古龙式 | 留白，短对白，危险感，物象， sudden reversal |
| 东野圭吾式 | 社会压力，误导线索，情感动机，最后的道德刺痛 |
| 刘慈欣式 | 宏大概念，冷峻因果，技术规则，人类尺度的牺牲 |
| 周星驰式 | 小人物受辱，荒诞喜剧，底层手艺，突然真情，公开翻身 |
| 韩剧式 | 高压亲密关系，身份错认，阶层羞辱，情绪拉扯，强结尾钩子 |

For living authors or distinctive contemporary voices, use the translated craft sliders only. Do not claim exact style imitation.

## Discussion Output

When the user gives a plot idea, output:

```text
我先给你几组经典桥段启发，咱们选完再组合成新故事。

1. A [经典信号]：[可复用功能]，适合做[效果]
2. B [经典信号]：[可复用功能]，适合做[效果]
3. C [经典信号]：[可复用功能]，适合做[效果]
4. D [经典信号]：[可复用功能]，适合做[效果]

我建议组合：[A + C + 一个反向处理]。
理由：[一句话]

你可以回复：按建议 / 1B 2D / 更狠 / 更悬疑 / 更现实。
```

Then build the attraction strategy and outline only after the user chooses or accepts the recommendation.

## Remix Checklist

Before drafting, verify:

- The inspiration works are reduced to functions, not copied scenes.
- At least three major elements changed: setting, profession/power system, relationship, object, stakes, ending.
- The story has one fresh object or rule that belongs only to the new version.
- The ending does not reproduce the reference work's unique twist.
- The prose uses translated craft features, not direct living-author imitation.
