# 1Panel Local App Packaging Checklist

## 安装前

- [ ] 应用 key 是英文小写
- [ ] 版本目录不带 `v`
- [ ] 顶层 `data.yml` 存在
- [ ] 顶层 `README.md` 存在
- [ ] `logo.png` 存在且大小合适
- [ ] 已准备安装参数和默认值
- [ ] 已确认数据卷路径

## 安装中

- [ ] `docker-compose.yml` 能启动
- [ ] 端口映射正确
- [ ] 环境变量正确注入
- [ ] 健康检查可通过
- [ ] 初始化脚本执行成功

## 安装后

- [ ] 1Panel 能识别该应用
- [ ] 页面能看到运行状态
- [ ] 日志正常
- [ ] 数据卷持久化正常
- [ ] 重启后状态正常

## 升级 / 卸载前

- [ ] 升级脚本不会改坏数据
- [ ] 卸载脚本不会误删持久化卷
- [ ] README 说明了边界
- [ ] 版本号递增方式明确

## 提交 AppStore 前

- [ ] 本地验证通过
- [ ] 目录结构符合规范
- [ ] 命名符合仓库约定
- [ ] 说明文档完整
- [ ] 图标和版本信息齐全
- [ ] 已准备 PR 分支
